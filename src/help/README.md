# src/help/

Stata help files (`.sthlp`) — one per public command. Written in SMCL.

## Style

- Open with a one-line title block, then `Syntax`, `Description`, `Options`, `Stored results`, `Examples`, `References`.
- Cross-link to related commands with `{help cmd:cmd}`.
- Cite the BCHS 2025 paper and the Christensen-Hansen guide in the `References` block.
